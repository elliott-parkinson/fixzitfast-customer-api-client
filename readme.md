# Fixzitfast public api client


A rest client for node.js and the browser that wraps and tests the publicly exposed customer api for the fixiztfast service.

## Requirements
  - node.js 13.9.x
  - npm 6.13.x
  

## Installation
```sh
$ npm install
```

## Running tests
```sh
$ npm run test
```
